# Electron + Vue3 + Vite - Builder

![Electron + Vue3 + Vite  ](screenshot.png)

## Quick Start

```sh
# clone the project
git clone https://github.com/laterly/electron-vue3-vite.git

# enter the project directory
cd electron-vue3-vite

# install dependency
npm install

# develop
npm run dev

#build

npm run build

#builder

npm run builder
```
